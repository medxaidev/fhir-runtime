# fhir-runtime

A lightweight, standards-compliant FHIR runtime engine for Node.js and TypeScript.

`fhir-runtime` provides core semantic capabilities for working with HL7® FHIR® resources, including parsing, validation, snapshot generation, and FHIRPath evaluation.

The project is designed as a pure runtime layer with no database or HTTP dependencies, making it suitable for embedding in servers, CLIs, web applications, or custom platforms.

---

## ✨ Features

- FHIR resource parsing
- StructureDefinition snapshot generation
- Profile-based validation
- FHIRPath evaluation
- Context loading (core spec and IG bundles)
- Pure TypeScript implementation
- No database or framework dependencies

---

## 📦 Installation

```bash
npm install fhir-runtime
```
